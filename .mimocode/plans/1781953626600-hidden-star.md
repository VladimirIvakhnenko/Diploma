# План доработки проекта TreeMaxFlow для публикации на GitHub

## Контекст
Проект — дипломная работа "О стационарных и динамических потоках в древовидных сетях" (ЮжФУ). Реализованы два алгоритма вычисления величины всплеска (burst) в древовидных сетях: базовый O(k·n) и оптимизированный O(n log n) с техникой Small-to-Large. Включает C++ ядро, интерактивную HTML/JS демо-визуализацию и Python-скрипты для построения графиков.

## 2. Подготовить README.md (русский, подробный)

**Файл:** `TreeMaxFlow/README.md` (создать с нуля)

Структура:
- **Заголовок** с названием проекта
- **Описание** — постановка задачи: стационарные и динамические потоки в древовидных сетях, определение burst
- **Математическая постановка** — определение burst-величины, формула через F^(d)_max
- **Алгоритмы**:
  - Базовый O(k·n): обход снизу вверх, перебор по уникальным глубинам
  - Оптимизированный O(n log n): Small-to-Large merge, хранение M[u] как map глубина→значение
  - Сравнительная таблица сложности
- **Структура проекта** — дерево файлов с описанием каждого
- **Сборка и запуск**:
  - Компиляция main.cpp (g++ -O2 -std=c++17)
  - Запуск эксперимента и получение результатов
  - Построение графиков (python plot.py / plot2.py)
- **Результаты экспериментов**:
  - Таблица результатов из experiment_results.txt
  - Встраивание графиков abs_time.png и asymptotic.png
- **Интерактивное демо** — описание demo.html, инструкция по использованию, скриншот
- **Зависимости** — g++17, Python 3 + matplotlib
